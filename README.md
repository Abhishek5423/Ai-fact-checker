# AI Report Fact Checker

This project extracts factual claims from market report PDFs and verifies them using live web search and AI.

## Features
- Upload a PDF report
- Extracts major factual claims
- Verifies claims as:
  - Verified
  - Inaccurate
  - False
- Displays sources and explanations

## Technologies
- Streamlit
- Google Gemini (Flash)
- Tavily Search API
- PyPDF
- Python
